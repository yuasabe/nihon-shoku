About Peer to Peer (P2P)

P2P, Peer to Peer, is the system to exchange files with users on the Internet.

There are systems which use P2P. (Ex. File sharing, IP Phone) In these days, you can use P2P to deliver videos.

Compared to Client-server model, load to server by this system is much smaller and anonymity is much higher. Because of its anonymity, watch to exchange of illegal data is very difficult. So it is truth that there are many such exchanges. Also the example like this - a founder of a file sharing system by P2P was arrested by suspect from complicity of piracy - has occurred. In Japan, Copyright Act amendment was enforced on January 1st in 2010, and then Japanese government became to be able to block the provider that exchanges illegal data from Internet. But the measure is not so effective.

As we expressed above, there is possibility to exchange files without notions in P2P. Therefore, there is also much possibility to catch computer virus. Most viruses caught via P2P function to steal users' private information. As measure, to open with inspection of manager, to ensure the use of antivirus software, to have users check uploaded files, were given. Especially, measure to virus must be necessary when we use P2P.
