Progress of program

Prologue

Compared with 50 years ago that computer was born, today's technical progress of computer is so wonderful.

Humans developed supercomputer which can manage huge data from vacuum tube computer which records data with punch card. And also we developed high-speed Internet network which can be connected by anyone and used by anyone from slow network that can be used by only specialist.

Such progress of computer goes on forever. We can say it not only in the ground of hardware but also software.

However, though convenience when we use computer has been much improved by one's technical progress, also risk has been increased very much.

Now, I will describe about the convenient point and dangerous point from progress of program.
