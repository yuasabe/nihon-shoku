About Complication of system

Cloud computing and maintenance of system bring us convenient and comfortable world. But how many users who use the system with true understanding are there? Cloud computing make difficult to manage data for users.

Please imagine the situation many kinds of information are in one place. Imagine the situation many sheets are on a desk. Of course they become a mess, and we become difficult to know where the file we want is. This is system failure by complex system. By complex system, sometimes we sent data to the place which is not concerned with, or upload to the place which can be seen by everybody. If the data was personal information, they are object of punishment by law (at least in Japan).

I think personal data is not used as something bad in many cases, but in some cases users become target of evil telephone, sending direct mail, etc. Cloud computing is much coming into fashion. Are you using it correctly?
