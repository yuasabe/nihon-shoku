About BBS

We will describe about BBS - Bulletin Board System - here.

BBS is the system which displays information that users submitted in turn with the system called "CGI"

We've made the trial version of BBS here.

*This is the TRIAL VERSION, so CGI is not used for it.

We can mention the name "2channel" as the example of the largest BBS in Japan. It is the BBS's' that founded in 1999. With the catchphrase "From 'hacking' to 'side dishes for tonight's dinner", there are many kinds of BBS where discuss many themes.

What can do in BBS is not only discussions but also submitting novels with the system that users contribute sentences, and pictures which used letter (AA).
