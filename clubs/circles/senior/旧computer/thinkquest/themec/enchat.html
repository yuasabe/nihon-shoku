About Chat

example of chat

We'll describe about "chat" here.

Chat is the system to talk with local people using a few sentences.

Generally, there is the place which called "Room" and users who came in talk freely. Also, they can set password and let particular users enter.

The advantage is to be able to communicate quickly compared with BBS. Because its amount of information is not so many, so sites can automatically renew the information from users in a few times.

As the meaning of "chat", it is the system whose nuance of "talking" is strong. We can do usual conversation with the letters by using "Chat".
